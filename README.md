# Test Task - Play to Max

Algorithm implementation for finding connected groups in a 2D grid.

## Live Demo

**[Play the game here](https://anton-kulchytskyi.github.io/test-task-play-to-max/)**

## Problem Statement

Given an MÃ—N grid where each cell contains an element, find all cells connected to a clicked cell that contain the same element (4-directional connectivity).
